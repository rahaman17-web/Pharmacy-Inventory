# Pharmacy Management System - Frontend

Built with React, Vite, and Tailwind CSS.

## Installation

```bash
cd Frontend
npm install
```

## Development

```bash
npm run dev
```

Access at `http://localhost:5173`

## Build

```bash
npm run build
```

## Features

- User Authentication & Dashboard
- Product Management
- Sales & Purchases
- Stock Reports
- Expenses & Returns
- Admin Controls
- Barcode Scanning
